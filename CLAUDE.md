# CLAUDE.md

Guidance for Claude Code in this repository.

## Style

Use caveman mode. Terse. Drop articles/filler/hedging. Fragments OK. Technical terms exact. Code blocks normal.

## Commands

```bash
bun run dev       # start dev server at localhost:4321
bun run build     # build static output to dist/
bun run preview   # preview the built site locally
```

No linting or test setup.

## Architecture

**Astro 6** static site (`output: 'static'`). File-based routing in `src/pages/`.

**Layout:** All pages wrap in `src/layouts/Base.astro` (accepts `title` prop). Renders `<head>`, global CSS vars, fixed `Navbar`.

**Design tokens:** CSS custom properties in `Base.astro` `<style is:global>` — use `--bg`, `--bg-2`, `--bg-3`, `--red`, `--text`, `--text-dim`, `--border`, `--accent`, `--font-mono`, `--navbar-h`. Never hardcode colors.

**URL utility:** `src/utils/url.ts` exports `url(path)` — prepends `BASE_URL` base path. Use for all internal `href` values in `.astro` frontmatter. In `<script>` tags use `import.meta.env.BASE_URL` directly.

**Pages:**
- `index.astro` — hero / landing
- `about.astro` — placeholder
- `projects.astro` — project cards with WASM renderers
- `contact.astro` — placeholder

**Components:**
- `Navbar.astro` — fixed top nav, uses `url()` for links
- `ProjectCard.astro` — card with title, description, repo link, canvas slot
- `WasmRenderer.astro` — fetches + runs WASM, renders to canvas via `requestAnimationFrame`. Lazy-loads via `IntersectionObserver`.

**Fonts:** Inter (sans) and JetBrains Mono (mono) from Google Fonts. `.mono` class applies `var(--font-mono)`.

## WASM

- Files go in `public/wasm/`
- Fetched at runtime via `${import.meta.env.BASE_URL}/wasm/<file>.wasm` — `BASE_URL` = `/portfolio` (no trailing slash), so leading `/` on path required
- `WasmRenderer.astro` waits for canvas visibility before fetching (IntersectionObserver)
- Current project: raytracer written in Zig (`WasmRenderer.wasm`)

## GitHub Pages deployment

- Deployed to `https://noknowledgeman.github.io/portfolio`
- `astro.config.mjs` must have `site: 'https://noknowledgeman.github.io'` and `base: '/portfolio'`
- CI uses Bun via `oven-sh/setup-bun@v2`, but Astro CLI requires Node ≥22 — `actions/setup-node@v4` with `node-version: 22` must run before Bun setup
- GitHub Pages source must be "GitHub Actions" in repo Settings → Pages
- `dist/` is generated by build, must not be committed
